# CareFlow

## Introduction
CareFlow is a healthcare workflow coordination app designed to help nurse managers, nurses, 
and CNAs track room assignments and patient care tasks during a shift.

The goal of CareFlow is to improve communication and prevent missed tasks by allowing staff to 
view and update assignments in real time from their mobile devices.

## Features
- Join shifts using a numeric join code
- Nurse managers assign rooms and tasks
- CNAs and Nurse's track tasks like meals, hygiene, and showers
- Real-time task updates
- Mobile-friendly workflow

## Technologies

1. Clone the repository

3. Allow Gradle to sync
4. Run the project on an emulator or Android device
